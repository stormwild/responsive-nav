# Responsive Navigation

## Development

### Setup 

#### Install Dev-Dependencies

```bash
npm install
```

Compile scss using node-sass

```bash
npm run build
```

Watch for changes

```bash
npm run watch
```

Ref: [Watch & Compile your Sass with npm](https://medium.com/@brianhan/watch-compile-your-sass-with-npm-9ba2b878415b#.7hnecf4wt)
